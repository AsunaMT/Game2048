程序运行说明
1.运行环境：
    应在配置有c++编译器的windows环境中运行；

2.如何编译和运行程序：
    在cmd中输入： g++ *.cpp  -o “文件名称” 以生成名称为“文件名称”的.exe文件，（也可不带该参数）
    运行.exe文件时可选的命令行参数： 
	-t  （测试模式，即最大值为64）；
	-s  “x”（x 为棋盘的维度，取值是2、3、4、5，若不在此范围内则会提示重新设定维度）每输入一个其他未定义的参数，
	        程序就会提示“You entered an invalid parameter!”，若未设定维度，则默认是4）；
	-i ” 文件名1“ -o ”文件名2“  （读入文件1 ，输出到文件2 ，如lab6要求）；
	-log 开启日志；
	-p    开启奖励模式。
